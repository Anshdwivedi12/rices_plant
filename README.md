## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Configuration

Update the WhatsApp number in `components/WhatsAppButton.tsx` and contact information in the relevant components.

## Build for Production

```bash
npm run build
npm start
```
